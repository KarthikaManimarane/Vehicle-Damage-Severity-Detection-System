# Vehicle-Damage-Severity-Detection-System
🚗 AI-powered vehicle damage detection and severity assessment system using computer vision, image classification, and damage analysis techniques.
******************************************************************************************************************************************************
An AI-powered vehicle damage detection and severity assessment system designed to identify damaged vehicle parts, classify damage severity, and generate structured damage reports.

## Features

* Vehicle Part Detection
* Damage Detection
* Damage Severity Classification
* Airbag Detection
* Multi-Image Analysis
* Streamlit-Based User Interface
* Automated Damage Report Generation

## Technology Stack

* Python
* Streamlit
* OpenCV
* NumPy
* Pandas
* Machine Learning Models
* Computer Vision

## Workflow

1. Upload vehicle images
2. Detect damaged components
3. Classify severity level
4. Generate structured damage report
5. Export analysis results
